# Moodle Task Hider

Moodleのダッシュボード（課題一覧・タイムライン）から、もう関係ない課題やテストを非表示にできるChrome拡張機能です。物理的に削除するわけではなく、拡張機能のポップアップからいつでも復元できます。

## 機能

- 各課題項目に「×」ボタンを追加し、クリックでその場で非表示
- 非表示にした項目は `chrome.storage.local` に保存され、ブラウザを閉じても保持される
- 拡張アイコンをクリックすると非表示リストが表示され、「復元」ボタンで元に戻せる

## インストール方法（開発者モード）

1. このリポジトリをクローンまたはダウンロード
2. Chromeで `chrome://extensions` を開く
3. 右上の「デベロッパーモード」をオンにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリックし、このフォルダを選択
5. Moodleのダッシュボードを開くと、各課題に非表示ボタンが表示される

## 仕組み

- `content.js` がMoodleのダッシュボード上の課題項目（`data-region="event-list-item"` など）を検出し、非表示ボタンを注入
- 非表示にした項目はリンクURLをキーにして保存するため、ページを再読み込みしても同じ項目を引き続き非表示にできる
- ポップアップ（`popup.html` / `popup.js`）は保存された非表示リストを表示し、復元操作を行う

## 注意

- MoodleのテーマやバージョンによってはダッシュボードのHTML構造が異なり、課題項目が検出できない場合があります。その場合は `content.js` の `ITEM_SELECTORS` を対象サイトのHTML構造に合わせて調整してください。
